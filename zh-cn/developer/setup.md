## 配置 Mohist 开发环境

> 可用版本 : **1.7.10**, **1.12.2**, **1.16.5**, **1.18.2**, **1.19.2**, **1.19.4**, **1.20**

### 步骤
* 克隆 Mohist 项目
    ```bash
    git clone https://github.com/MohistMC/Mohist.git -b <mc-version>
    ```
* 配置
  * 在 Linux 环境下配置:  
    `bash gradlew setup`
  * 在 Windows 环境下配置:  
    `gradlew.bat setup`
