# MohistMC - Wiki

欢迎来到 MohistMC 的 wiki！在这里，您可以找到关于 Mohist 的一些信息以及如何配置它。
> 本页面正在完善中，您也可以参与编写！

Mohist 是一款实现了 Bukkit、Spigot 和 Paper API 的 Forge 服务器软件。这个项目的目的是为了让**模组和插件可以一起使用**。目前，Mohist 提供了7个版本（1.7.10、1.12.2、1.16.5、1.18.2、1.19.2、1.19.4、1.20）。

## 关于Mohist
未来将会有更多版本推出。

- **Mohist 1.20** 积极更新中。
